La página principal tiene la función de guiar y dirigir al usuario a otras páginas del sitio web

La estructura física de un sitio web es la forma en que se almacenan los distintos recursos (ficheros) que forman un sitio web en el sistema de archivos del servidor web
El lenguaje HTML consta de una serie de etiquetas o marcas (tags)
La mayoría de las etiquetas aparecen por parejas (códigos pareados), siendo una de inicio (apertura) y otra de (cierre)
Todas las etiquetas comienzan con el símbolo < y terminan con el símbolo>
Las etiquetas de cierre tienen el mismo nombre que las de inicio, pero van precedidas del símbolo /
Una etiqueta puede poseer varios atributos a los que hay que asignar valor
  • 1. Un usuario solicita una página HTML a un servidor web

  • 2. Este envía la página tal cual

  • 3. El explorador recibe la página

  • 4. Interpreta las etiquetas que la página contiene

  • 5. Muestra al usuario el resultado final
