Computadora

Del Latín "computare" , que significa calcular o sumar

Es un dispositivo electrónico usado para procesar datos

La mayor parte de las tareas se realizan en la unidad de procesamiento central (CPU)

Elementos básicos

  • Software
  • Hardware

Sofware

Conjunto de programas, instrucciones y reglas informáticas para ejecutar ciertas tareas en una computadora

-RAE

Las instrucciones que entiende una computadora, deben ser dadas en binario

Hardware

Dispositivos electrónicos relacionados entre sí, cuya función es controlar las operaciones a realizar, la entrada y la salida de datos.

Los componentes más básicos son:

  • CPU
  • Tarjeta Madre
  • Memoria RAM
  • Disco Duro
